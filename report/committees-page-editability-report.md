# Committees Page Editability Report

## College Development Committee

- Editable Content:
  - Responsibilities
  - Members (Post, Name)
  - Key roles (Chairman, Secretary)
- Character Limits:
  - Responsibility text 80-120 characters
- Item Limits:
  - Responsibilities max 5-6
  - Members max 10
- Reason:
  - Maintains readability and proper table alignment

---

## IQAC

- Editable Content:
  - Objectives
  - Committee members
  - Reports (Year, PDF)
- Character Limits:
  - Objective text 80-120 characters
- Item Limits:
  - Objectives max 5-6
  - Reports max 5
- Reason:
  - Prevents overcrowding of report sections

---

## Anti-Ragging Committee

- Editable Content:
  - Objectives
  - Members (Name, Designation, Contact)
- Character Limits:
  - Objective text 80-120 characters
- Item Limits:
  - Objectives max 5-6
  - Members max 8
- Reason:
  - Keeps layout clean and structured

---

## Grievance Redressal Committee

- Editable Content:
  - Objectives

- Item Limits:

## Student Grievance Redressal Committee (SGRC)

## Equal Opportunity Cell (PDF-based page)

- Editable Content:
  - PDF file (main document content - edited separately and replaced)
  - PDF URL/link
- Character Limits:
  - Not applicable
- Item Limits:
  - Usually 1 main PDF (max 2-3 if needed)
- Reason:
  - Content cannot be edited on the webpage; it is updated by modifying and replacing the PDF file

---

## SEDG Cell (PDF-based page)

- Editable Content:
  - PDF file (main document content - edited separately and replaced)
  - PDF URL/link
- Character Limits:
  - Not applicable
- Item Limits:
  - Typically 1 main PDF
- Reason:
  - Content is managed through the document file, ensuring simple and direct access without webpage editing
